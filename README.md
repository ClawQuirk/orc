# Orc

Built on [ClawQuirk Canvas](https://github.com/ClawQuirk/clawquirk-canvas) — a self-hosted web app with an integrated terminal side panel.

## Getting Started

```bash
# Check prerequisites and install dependencies
npm run setup

# Start both frontend and backend
npm run dev
```

Open http://localhost:5173 in your browser.

## Custom Ports

```bash
PORT=8080 BACKEND_PORT=4001 npm run dev
```
